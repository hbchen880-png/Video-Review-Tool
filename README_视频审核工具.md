# 视频审核工具（界面优化版）

本次按你的 4 个界面意见做了调整：

1. 顶部工具区改成紧凑工具栏，不再保留大块空白区域。
2. 删除波形下方那一整块状态/按钮区域，界面更干净。
3. 音频波形区域保留，并支持通过右侧“眼睛”按钮隐藏/显示；视频区和波形区仍可通过中间分隔条调整高度，比例会记住。
4. 进度条与音频波形对齐为同一宽度，时间显示移动到进度条中间。
5. 左侧队列 / 右侧记录 / 下方时间轴区域增加更细的最小宽高限制，小屏下更稳，拖拽更顺手。
6. 波形显隐按钮改成更正式的图标按钮，不再使用表情字符。
7. 修复了部分环境下打包后启动报错：`QSizePolicy` 没有 `Fixed` 属性的问题。
8. 设置弹窗新增“重置布局比例”按钮，可一键恢复默认分区比例，并自动保存。

当前保留功能：
- 左侧待审核队列 / 中间视频 / 下方时间条+波形 / 右侧审核记录
- 目录与快捷键设置
- Space 暂停/继续
- 拖动进度条时暂停并显示当前位置画面
- I 设置起点，O 裁剪 I~O 区间到通过目录
- V 通过，N 不通过，Y 返回上一条，E 结束审核
- 布局比例自动保存，下次启动自动恢复
- 设置弹窗内可一键“重置布局比例”

## 使用
直接运行 `video_review_app.py`。

## 打包 exe
使用 `build_exe_fixed_ascii.bat`。

# 视频审核工具 - 最终稳定版 EXE 打包（无感切换版）包（切换更顺畅版）

## 目录说明
- `video_review_app_final_stable.py`：最终稳定版主程序
- `video_review_tool_final_stable.spec`：PyInstaller 打包配置
- `build_final_stable_exe.bat`：一键打包脚本
- `requirements_build.txt`：打包依赖
- `app_logo.png` / `app_logo.ico`：程序图标

## 这版额外修复
- 修复切换视频时 Windows 黑色运行弹窗一闪而过的问题
- 优化视频切换流畅度：
  - 波形结果增加缓存
  - 提前预加载下一个视频的波形
  - 参考图匹配结果增加缓存
  - 参考图图片增加缓存
  - 参考图索引预处理，减少切换时卡顿

## 已包含功能
- 右侧参考图片区替换原通过区
- `Z` 键切换当前产品参考图
- 切换后记住该产品默认参考图
- 设置中可配置产品库目录
- 对网络共享目录不可访问、账号锁定、用户名密码错误做容错
- 源目录 / 产品库 / 通过目录 / 不通过目录异常时不再启动崩溃

## 打包方法
1. 把整个文件夹放到本地磁盘路径下
2. 双击 `build_final_stable_exe.bat`
3. 等待打包完成
4. 生成结果在：`dist/视频审核工具稳定版/视频审核工具.exe`

## 建议
- 尽量不要把打包目录放在中文过深路径、桌面同步盘、NAS 映射目录里
- 如果你的源目录或产品库目录是共享盘，首次运行前先确认 Windows 已经能正常访问该共享路径
- 若之前存在旧版 `review_config.json`，这版可继续读取；如配置异常，可删掉后重开让程序重新生成

## 默认快捷键
- `V` 通过
- `N` 不通过
- `Y` 返回上一条
- `E` 结束审核
- `Space` 暂停/继续
- `I` 设置裁剪起点
- `O` 裁剪并通过
- `Z` 切换参考图

## 备注
当前打包方案使用 **onedir** 模式，稳定性比单文件更高，适合作为最终交付版。


## 本版新增优化
- 选择后切下一个视频改为事件循环后再切源，界面更顺
- 仅预热下一个/下两个视频文件与参考图，不再激进预取下一条波形，减少播放期卡顿
- 队列刷新改为局部刷新，视频很多时切换更流畅
- ffmpeg 子进程继续隐藏黑框
